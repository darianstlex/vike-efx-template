## Getting Started

Development:

```bash
npm i / npm ci
npm run dev
```

## Tests

To run unit tests

```bash
npm run unit
npm run unit:open
npm run unit:coverage
```

To run functional tests

```bash
npm run func
npm run func:open
npm run func:coverage

npm run test:coverage
```

To run all tests wth coverage

```bash
npm run test:coverage
```
